Test Case DISPLAY-0030
Title If JavaScript is enabled, the user has the ability to see more detailed information.
Objective

This test case is intended to verify the following implied or specified requirements:

  • When an HTTP client performs a GET to the site URL, the server shall provide an index.html page that has the following information:
  • The page shall have a title that includes the provisioned location name.
  • The page shall have the temperature.
  • The page shall have the wind speed and direction.
  • The page shall have a "current conditions" statement (eg cloudy, sunny&#8230;).
  • The page shall have a timestamp displayed in the local time zone of the configured weather location.
  • The page shall have weather alerts issued by the nation weather service (NWS).
  • The page shall have forecasted temperature, wind, and condition information.
  • The client user interface shall only have controls that are associated with changing the presentation of the amount of information.
  • The initial display of a client query shall be minimal.
  • By selecting a checkbox or checkboxes, the user shall be able to see more information if Javascript is enabled.

This test case assures that when JavaScript is enabled, the user can see more detailed information.
Setup
  • Test case INST-0010 has been run.
Actions
  1. Using a browser, access the site.
  2. Verify that the current temperature is displayed.
  3. Verify that the current wind speed and direction are displayed.
  4. Verify that the current conditions are displayed.
  5. Verify that future (forecast) weather information is displayed.
  6. Check if there are alerts (there may not be any)
  7. Verify there is a timestamp on the page that corresponds to the current local time.
  8. Verify that the page has a title that refers to the location provisioned.
  9. Confirm that the forecast table has a checkbox that is labeled "display all" and the box is unchecked.
  10. Press the checkbox.
  11. Verify that more forecast information is displayed in the table and the box is now checked.
  12. Press the checkbox.
  13. Verify that the table returns to the original state.
  14. Verify that there is a temperature graph showing the temperature throughout the next 24 hours.
Expected The displayed page will have the required displays and that with JavaScript, the amount of forecast data can be controlled and the temperature can be plotted.
Results PASSED
