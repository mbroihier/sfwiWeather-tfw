Display

This test category contains test case meant to verify that the display requirements are met. These tests use a client that understands HTML5 and JavaScript. Most modern browsers will do that.

  1. DISPLAY-0010 - Display contains current conditions and forecast information
  2. DISPLAY-0020 - Display contains current conditions and forecast information with JavaScript off on browser
  3. DISPLAY-0030 - If JavaScript is enabled, the user has the ability to see more detailed information.
  4. DISPLAY-0040 - Display contains current conditions and forecast information at a location with alerts
  5. DISPLAY-0050 - Display uses HTML5
