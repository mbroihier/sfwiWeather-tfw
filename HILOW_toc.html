High Low

This test category tests the ranges of temperature and wind. Checks exist within the server to protect itself from data that is not normal. That data is rejected. This test category was built by the API emulator and is meant to be run using the test driver. From within the sfwiWeather-tfw directory, executing node driver.js HILOW will execute this series of tests.

  1. HILOW-0000 - Test category setup
  2. HILOW-0010 - Ever increasing wind and temperature
  3. HILOW-0020 - Ever increasing wind and temperature
  4. HILOW-0030 - Temperature exceeds range of temperature - high
  5. HILOW-0040 - Temperature exceeds range of temperature - low
  6. HILOW-0050 - Wind speed is negative
  7. HILOW-0060 - Wind speed is too fast
  8. HILOW-0070 - Wind speed and temperature are just within tolerance - high side
  9. HILOW-0080 - Wind speed and temperature are just within tolerance - low side
