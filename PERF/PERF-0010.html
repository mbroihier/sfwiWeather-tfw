Test Case PERF-0010
Title Server responds in under three seconds.
Objective

This test case is intended to verify the following implied or specified requirements:

  • The client shall be able to receive a response from the server in under three seconds when there are no other clients being served.

This test case times the end to end response to a query and verifies that, on an unloaded system (this appliance is only meant to run on a small home network), the response time of the query is under three seconds.
Setup
  • This test case assumes INST-0010 has been run.
Actions
  1. From another computer on the sfwiWeather's network that is running Linux, run the following commands on a single line:
    • date ; curl -X GET "http://sfwiWeather:3000" ; date
  2. Measure the difference between the time stamps.
  3. Repeat this 9 more times.
  4. Verify that all time differences are less than three seconds.
Expected All measured time deltas will be less than three seconds.
Results PASSED
Cleanup
